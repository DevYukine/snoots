snoots

A modern, fully-featured, strongly-typed reddit api.

WARNING

This project is in early alpha. Things can (and likely will) change at any time for any reason, and large parts of the api are not yet supported. If you need a stable reddit api today, use snoowrap.

If you want to help guide the future of the project towards v1.0.0, please check out this discussion!

Installation

 
npm i snoots

Note: Node 14+ is required.
